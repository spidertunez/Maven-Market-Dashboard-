📊 Maven Market — Power BI Dashboard
Overview
This project is an interactive Power BI report for Maven Market, a retail company. It provides a comprehensive analysis of sales, profit, customers, and returns across multiple integrated pages.

Report Structure
The report contains 5 main pages:

1. 🏠 Welcome Page
An introductory page with images, titles, and navigation buttons linking to all other pages.

2. 📈 Overview
Purpose: Display key performance indicators for the company's overall performance.

3. 📦 Products
Purpose: Analyze product and brand performance.

4. 👥 Customers
Purpose: Analyze the customer base and purchasing behavior.

5. 🔄 Returns
Purpose: Track and analyze product returns across regions and brands.
Visual / KPITypeReturns Trend Over TimeLine ChartReturns ComparisonBar ChartReturn Rate by Product BrandBar ChartQuantity Returned by GenderDonut ChartReturn per TransactionCardQuantity ReturnedCardReturns CountCardReturn RateCardFilters: Sales Region & Product NameSlicer

🎨 Design & Theme

Theme: Maven Market Light (custom theme)
Assets: Animated GIFs and SVG icons used to enhance user experience
Logo: Maven Market logo included across all pages


🛠️ Requirements

Power BI Desktop (or Power BI Service for sharing)
Data source files connected to Maven Market datasets
